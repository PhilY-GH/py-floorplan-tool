快捷键
框选可多选；Shift+点击可增减选择
点击画布左上角状态栏中的“已选 x 个对象”可开启 / 关闭多选模式
绘制线、箭、墙、尺时，按 Shift 可锁定 0° / 45° / 90°
拖动方 / 圆角点时，按 Shift 可保持比例
绘制区域时，按 Enter 完成闭合；按 Esc 取消绘制
拖动节点或控制点时，按 Esc 可取消本次调整
拖动对象时，Shift 可锁定水平 / 垂直；Alt / Option 可复制拖动
Alt / Option + Shift 拖动：复制并水平 / 垂直约束移动
文字编辑框：Enter 换行；Ctrl / Command + Enter 确定；Esc 取消
方向键微调 1 px；Shift+方向键移动 10 px，不受吸附限制
Ctrl+C / Ctrl+V 复制粘贴
Ctrl+G 组合；Ctrl+Shift+G 取消组合
